# SMS Viewer

Dette værktøj tillader dig at se dine SMS-samtaler fra en XML-backup-fil (f.eks. fra Android-enheder).

## Brug

1. Åbn `index.html` i din browser.
2. Klik på "Vælg SMS backup XML fil" og vælg din XML-fil med samtaler.
3. Gennemse samtalerne i venstre panel og klik på en for at se beskederne.

## Filstruktur

- `index.html` - Hoved-HTML-fil
- `styles.css` - CSS-styling
- `script.js` - JavaScript-funktionalitet

## Funktioner

- Visning af alle samtaler med antal beskeder
- Visning af beskeder i en samtale med forskellige farver for sendte og modtagne beskeder
- Søgning i alle beskeder og kontaktnavne
- Fremhævning af søgeord i beskeder
- Visning af dato og tid for hver besked
